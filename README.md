# 🌪 NDS Disaster Predictor

سكريبت لـ **National Disaster Survival** يعرض لك الكارثة القادمة قبل ما تبدأ.

## الاستخدام

```lua
loadstring(game:HttpGet('https://raw.githubusercontent.com//main/predict.lua'))()
```

## المميزات

- 🔍 يكتشف الكارثة تلقائياً من ملفات اللعبة
- ⚡ يدعم: برق، إعصار، فيضان، زلزال، نووية، نيزك، ثلج، حريق، رمال
- 🎨 UI أنيق يظهر اسم الكارثة ولونها
- ⏱ يعرض الوقت منذ اكتشاف الكارثة
- 🔒 يشتغل فقط مع اللاعب المحدد

## ملاحظات

- يفحص workspace و ReplicatedStorage كل ثانية
- قابل للتحريك (Draggable)
